# 🚀 LeetCode Practice Solutions

This repository contains my solutions to various problems from Leetcode. 
The goal of this project is to improve problem-solving skills, strengthen Data Structures & Algorithms (DSA), and prepare for technical interviews.


Each folder contains solutions related to that specific topic.
